Escribir una clase TorreDeControl que modele el trabajo de una torre de control de un aeropuerto con una pista de aterrizaje. Los aviones que están esperando para aterrizar tienen prioridad sobre los que están esperando para despegar. La clase debe funcionar conforme al siguiente ejemplo: Podés usar la implementación de Cola ( cola = Cola()) 

    >>> torre = TorreDeControl()
    >>> torre.nuevo_arribo('AR156')
    >>> torre.nueva_partida('KLM1267')
    >>> torre.nuevo_arribo('AR32')
    >>> torre.ver_estado()
    Vuelos esperando para aterrizar: AR156, AR32
    Vuelos esperando para despegar: KLM1267
    >>> torre.asignar_pista()
    El vuelo AR156 aterrizó con éxito.
    >>> torre.asignar_pista()
    El vuelo AR32 aterrizó con éxito.
    >>> torre.asignar_pista()
    El vuelo KLM1267 despegó con éxito.
    >>> torre.asignar_pista()
    No hay vuelos en espera.
